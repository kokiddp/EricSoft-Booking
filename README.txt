This plugin integrates a EricSoft Booking reservation form.

== Description ==

Integrate EricSoft Booking reservation painlessly in your website.

The plugin is developed using AngularJs, Moment.js and Lodash.
JavaScript must be activated for the plugin to work.

== Installation ==

As simple as it gets:

1. Upload the plugin
2. Activate it
3. Configure it by setting the option fields under Settings -> EricSoft Booking
4. Place [veb_display_form] shortcode wherever you want
